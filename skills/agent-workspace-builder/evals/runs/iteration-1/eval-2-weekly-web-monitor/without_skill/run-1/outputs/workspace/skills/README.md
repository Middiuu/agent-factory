# Skills locali

Non è necessaria alcuna skill locale per questo workspace. Configurazione, fetch HTTP, hashing, diff, persistenza, cadenza e CLI sono coperti dalla standard library Python e dal codice versionato in `src/`.

La directory è mantenuta per rendere esplicita questa decisione. Non aggiungere integrazioni di notifica, credenziali o scheduler sotto forma di skill: sono fuori ambito e richiedono una decisione separata dell'operatore.
